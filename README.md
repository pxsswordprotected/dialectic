# Dialectic

A philosophy learning platform built on first principles from the science of learning. Dialectic applies techniques like spaced repetition, mastery-based progression, knowledge graphs, and active retrieval practice to teach propositional logic.

Built as an undergraduate independent study project under Dr. Xing Liu.

## How It Works

Courses are structured as a knowledge graph of topics that depend on one another. Each topic contains two phases: a slide-based lesson broken into small, focused chunks to manage cognitive load, followed by practice questions that test recall with immediate feedback. Students must demonstrate mastery before advancing to the next topic. Completed topics enter a spaced repetition review schedule (SM-2 algorithm) that brings them back at timed intervals to keep knowledge permanent.

## Tech Stack

- **Framework:** Next.js (App Router), React, TypeScript
- **Database:** Supabase (PostgreSQL)
- **ORM:** Drizzle
- **Styling:** Tailwind CSS
- **Hosting:** Vercel
